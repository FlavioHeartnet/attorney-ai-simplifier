

## Description

This is project intented to easy lawer customers, by providing and API get legal text related to the client's case and rewrite the text in a simple way that non-lawyer clients can understand.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
